## 1. 프로젝트 개요 (Overview)

**Looky**는 저해상도 CCTV·영상으로부터 **고해상도(HR) 영상을 복원**하고,  
복원된 영상에서 **특정 실종자를 정확히 탐지·식별**하기 위한  
**AI 기반 영상 복원(Super-Resolution) + 얼굴 인식 파이프라인 시스템**이다.

### 🎯 프로젝트 목표
- 화질이 매우 낮은 실종자 수색 환경(CCTV, 원거리 촬영 영상)에서도  
  **식별 가능한 수준의 영상 품질 확보**
- 단순 화질 개선을 넘어, **실제 실종자 탐지·식별에 기여하는 복원 파이프라인 구현**
